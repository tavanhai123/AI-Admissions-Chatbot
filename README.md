# AI Admissions Chatbot (NLP + Neural Network)

## Giới thiệu
Chatbot hỗ trợ tư vấn tuyển sinh dựa trên **xử lý ngôn ngữ tự nhiên (NLP)** và **mạng neural network**.  
Bot được huấn luyện trên dữ liệu thu thập từ Facebook/website, có khả năng phân loại câu hỏi theo intent và phản hồi tự động.

---

## Tính năng chính
- Thu thập dữ liệu (web crawling → CSV)  
- Tiền xử lý NLP: tokenization, stemming, stopwords  
- Feature: Bag-of-Words  
- Huấn luyện mô hình neural network (TensorFlow + TFLearn)  
- Lưu/tải mô hình với pickle hoặc model.save  
- Chat qua terminal hoặc module inference
